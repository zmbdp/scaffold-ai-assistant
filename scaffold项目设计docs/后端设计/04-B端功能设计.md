# 📄 B端功能设计文档

> **文档说明**：本文档详细描述 Scaffold AI Assistant B端（管理端）的功能设计。
> **复用说明**：用户管理（sys_user 表 CRUD）复用现有 `zmbdp-admin-service` 的实现。操作日志使用现有 `@LogAction` 注解自动记录。
> **重要说明**：脚手架**未实现**角色管理（无 sys_role 表）和细粒度权限控制（无 sys_permission 表），sys_user.identity 字段仅作标识存储。B端接口的访问控制通过扩展 Gateway AuthFilter 实现（详见 07-项目架构设计.md 7.2.24节）。

## 4.2 功能模块详细设计

### 4.2.1 知识库管理模块

**功能描述**：管理知识源、上传文档、触发知识同步、查看文档状态

**子功能**：

| 子功能 | 描述 |
|-------|------|
| **知识源管理** | 查看、新增、编辑、删除知识源（目录路径、类型等） |
| **文档上传** | 上传新文档到知识库 |
| **知识同步** | 手动触发知识同步，更新向量数据库 |
| **文档状态** | 查看文档索引状态、更新时间、大小等 |
| **文档搜索** | 搜索知识库中的文档 |

**API接口**：

| HTTP方法 | 路径 | 说明 |
|---------|------|------|
| GET | `/knowledge/sources` | 获取知识源列表 |
| POST | `/knowledge/sources` | 新增知识源 |
| PUT | `/knowledge/sources/{id}` | 更新知识源 |
| DELETE | `/knowledge/sources/{id}` | 删除知识源 |
| POST | `/knowledge/sync` | 触发知识同步 |
| GET | `/knowledge/documents` | 获取文档列表 |
| GET | `/knowledge/documents/{id}` | 获取文档详情 |
| DELETE | `/knowledge/documents/{id}` | 删除文档 |

### 4.2.2 AI配置管理模块

**功能描述**：配置AI模型参数、API Key等

**子功能**：

| 子功能 | 描述 |
|-------|------|
| **模型配置** | 选择AI模型（通义千问/OpenAI） |
| **API Key管理** | 配置和管理API Key（加密存储） |
| **参数配置** | 调整temperature、max-tokens等参数 |
| **Embedding配置** | 配置向量嵌入模型和参数 |

**配置项**：

| 配置项 | 说明 | 示例值 |
|-------|------|--------|
| `spring.ai.dashscope.api-key` | 通义千问API Key | - |
| `spring.ai.dashscope.chat.model` | 聊天模型 | `qwen-max` |
| `spring.ai.dashscope.chat.options.temperature` | 温度参数 | `0.7` |
| `spring.ai.dashscope.chat.options.max-tokens` | 最大token数 | `4096` |
| `spring.ai.dashscope.embedding.model` | 嵌入模型 | `text-embedding-v1` |

**API接口**：

| HTTP方法 | 路径 | 说明 |
|---------|------|------|
| GET | `/ai/config` | 获取AI配置 |
| PUT | `/ai/config` | 更新AI配置 |
| GET | `/ai/models` | 获取可用模型列表 |
| POST | `/ai/test` | 测试AI连接 |

### 4.2.3 使用统计模块

**功能描述**：查看系统使用情况、对话统计等，结合Grafana+LangSmith实现AI调用追踪和评估

**子功能**：

| 子功能 | 描述 | 数据来源 |
|-------|------|---------|
| **对话统计** | 总对话次数、今日对话次数、趋势图 | 数据库聚合查询 |
| **热门问题** | 统计最常被问的问题TOP10 | 数据库聚合查询 |
| **用户活跃度** | 活跃用户数、使用频次 | 数据库聚合查询 |
| **工具使用统计** | 各Agent工具的调用次数 | 数据库聚合查询 |
| **响应时间统计** | AI响应时间分布 | Grafana监控 |
| **AI调用追踪** | 查看每次AI调用的完整链路（Prompt、LLM响应、工具调用） | LangSmith |
| **Token消耗统计** | AI调用的token消耗、成功率、平均耗时 | LangSmith |

**统计指标**：

| 指标 | 说明 |
|-----|------|
| `totalConversations` | 总对话次数 |
| `todayConversations` | 今日对话次数 |
| `avgResponseTime` | 平均响应时间 |
| `topQuestions` | 热门问题列表 |
| `activeUsers` | 活跃用户数 |
| `tokenUsage` | Token消耗统计 |
| `successRate` | AI调用成功率 |

**API接口**：

| HTTP方法 | 路径 | 说明 |
|---------|------|------|
| GET | `/statistics/conversations` | 获取对话统计 |
| GET | `/statistics/questions` | 获取热门问题 |
| GET | `/statistics/users` | 获取用户统计 |
| GET | `/statistics/tools` | 获取工具使用统计 |
| GET | `/statistics/ai-metrics` | 获取AI调用指标 |

**数据来源**：
- **API调用统计**：后端提供统计API接口，供前端调用展示（对话统计、热门问题、用户统计、工具使用统计）
- **Grafana监控**：通过Prometheus采集应用指标（请求量、响应时间、错误率），用于系统级监控告警
- **LangSmith追踪**：集成Spring AI的LangSmith Tracer，追踪每次AI调用的完整链路，支持调试和评估

### 4.2.4 系统管理模块

**功能描述**：用户管理、操作日志、系统状态等

**子功能**：

| 子功能 | 描述 |
|-------|------|
| **用户管理** | 用户列表、新增、编辑、删除、禁用（复用脚手架 sys_user 表） |
| **操作日志** | 查看管理员操作记录（复用 `@LogAction` 注解） |
| **系统状态** | 查看系统健康状态、组件状态 |

**用户身份标识**（sys_user.identity 字段，仅作标识，无细粒度权限校验）：

| identity 值 | 说明 | 当前权限控制 |
|------------|------|-------------|
| `super_admin` | 超级管理员 | 仅标识，无额外权限逻辑 |
| `platform_admin` | 平台管理员 | 仅标识，无额外权限逻辑 |

> **注意**：脚手架未实现角色体系（无 sys_role 表）和细粒度权限控制（无 sys_permission 表）。如需实现 RBAC，需作为 v2.0 功能新增开发。

**API接口**：

| HTTP方法 | 路径 | 说明 |
|---------|------|------|
| GET | `/knowledge/logs` | 获取知识库操作日志 |
| GET | `/system/health` | 获取系统健康状态 |

> **说明**：用户管理接口已在 `zmbdp-admin-service` 中实现，本模块不再重复开发。AI 模块仅新增知识库操作日志查询接口。

### 4.2.5 工具管理模块

**功能描述**：管理Agent工具的启用/禁用、配置工具参数

**子功能**：

| 子功能 | 描述 |
|-------|------|
| **工具列表** | 查看所有Agent工具 |
| **工具状态** | 启用/禁用工具 |
| **工具配置** | 配置工具参数（如文件路径白名单） |
| **工具测试** | 测试工具功能 |

**工具列表**：

| 工具名称 | 默认状态 | 描述 |
|---------|---------|------|
| ReadFileTool | 启用 | 读取指定文件内容 |
| SearchCodeTool | 启用 | 搜索代码中的类/方法 |
| ListDirTool | 启用 | 列出目录结构 |
| SearchInFileTool | 启用 | 在文件中搜索关键字 |
| NacosConfigTool | 启用 | 查询Nacos配置项 |
| CompareConfigTool | 启用 | 对比不同环境配置 |
| PreDeployCheckTool | 启用 | 部署前检查清单 |

**API接口**：

| HTTP方法 | 路径 | 说明 |
|---------|------|------|
| GET | `/tools` | 获取工具列表 |
| PUT | `/tools/{name}` | 更新工具配置 |
| POST | `/tools/{name}/test` | 测试工具 |

## 4.3 B端功能优先级

| 优先级    | 功能模块   | 理由                |
| ------ | ------ | ----------------- |
| **P0** | 知识库管理  | 基础功能，没有知识库就没有AI回答 |
| **P0** | AI配置管理 | 基础功能，必须配置才能使用AI   |
| **P1** | 使用统计   | 了解系统使用情况          |
| **P1** | 工具管理   | 管理Agent工具         |
| **P2** | 系统管理   | 用户管理（复用脚手架）、操作日志         |
| **P2** | 数据管理   | 备份恢复等高级功能         |
